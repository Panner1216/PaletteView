# RectanglePickerView
## 一个矩形的色盘
    可以设置色盘的宽、高，拖拽圆的半径、颜色等
### xml中
    picker_width_rect：色盘宽度
    picker_height_rect：色盘高度
    indicator_width_rect：拖拽圆半径
    indicator_color_rect：拖拽圆颜色
### 代码
    setWidth：色盘宽度
    setHeight：色盘高度
    setTouchViewColor：拖拽圆颜色
    setTouchViewRadius：拖拽圆半径
